Components and interfaces
#########################

.. py:module:: amaranth.lib.wiring

The :mod:`amaranth.lib.wiring` module provides a way to describe the interfaces between components and connect them to each other in a reliable and convenient manner.

.. testsetup::

   from amaranth.lib import wiring
   from amaranth.lib.wiring import In, Out
